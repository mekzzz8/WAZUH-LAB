# WAZUH-LAB
Implementación de un entorno SIEM/XDR con Wazuh para la monitorización centralizada de eventos y la detección de amenazas en tiempo real. Configuración de agentes y reglas personalizadas alineadas con MITRE ATT&amp;CK para identificar vulnerabilidades y cambios de integridad en archivos.
